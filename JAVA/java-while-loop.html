Repeating Actions with Loops

Imagine you need to tell a robot to take 10 steps forward. Instead of writing the "take a step" command 10 times, you can use a loop to repeat the command automatically.

The `while` loop is a fundamental loop in Java that repeats a block of code as long as a specified condition remains `true`.

The `while` Loop

The `while` loop checks its condition *before* it runs the code inside. If the condition is true, the code block executes. This process repeats until the condition becomes false. Let's create a countdown for a rocket launch.

int countdown = 5;

while (countdown > 0) {
  System.out.println(countdown + "...");
  countdown--; // Decrease countdown by 1 each time
}

System.out.println("Liftoff!");
Output:
5...
4...
3...
2...
1...
Liftoff!

The `do/while` Loop

The `do/while` loop is a variation of the `while` loop. The key difference is that the `do/while` loop will execute the code block **at least once**, before it checks if the condition is true. This is useful for situations where you always want the action to happen at least one time, like showing a menu.

import java.util.Scanner;

class Menu {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    int choice;

    do {
      System.out.println("\n--- MENU ---");
      System.out.println("1. Start Game");
      System.out.println("2. Load Game");
      System.out.println("3. Exit");
      System.out.print("Enter your choice: ");
      choice = input.nextInt();
    } while (choice != 3);

    System.out.println("Thanks for playing!");
  }
}
Output (will keep showing the menu until user enters '3'):
--- MENU ---
1. Start Game
2. Load Game
3. Exit
Enter your choice:
