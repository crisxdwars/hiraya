Labeling Your Data

Variables are like labeled boxes where you can store information. Before you can store something, you have to create a box and give it a name. In Java, you also have to tell the box what type of information it will hold.

Creating and Using Variables

To create a variable, you declare its type, give it a name, and assign it a value using the `=` sign. Let's create a user profile:

// Storing user profile information
String username = "Alex";
int userAge = 25;
char userInitial = 'A';
boolean isSubscribed = true;

System.out.println("Username: " + username);
System.out.println("Age: " + userAge);
Output:
Username: Alex
Age: 25

The `final` Keyword: Unchangeable Variables

Sometimes you have a value that should never change, like a user's date of birth or a mathematical constant. You can use the `final` keyword to create a constant variable. Trying to change it later will cause an error.

final String DATE_OF_BIRTH = "2000-01-20";

// The following line would cause a compiler error:
// DATE_OF_BIRTH = "2001-05-10";

System.out.println("User's Date of Birth: " + DATE_OF_BIRTH);
Output:
User's Date of Birth: 2000-01-20
