Making Your Programs Interactive

To make your programs more engaging, you need to be able to get input from the user. Java provides a tool called the `Scanner` class for this purpose.

To use it, you first need to `import` it at the top of your file. This is like telling your program you're going to use a specific tool from your Java toolbox.

import java.util.Scanner;  // Import the Scanner class

Reading Different Types of Input

The `Scanner` has different methods to read different data types. For example, `nextLine()` reads text, and `nextInt()` reads a whole number.

Let's create a simple character creation prompt:

import java.util.Scanner;

class CharacterCreator {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);
    
    System.out.println("--- Create Your Character ---");
    
    System.out.print("Enter your character's name: ");
    String name = input.nextLine();

    System.out.print("Enter your character's level: ");
    int level = input.nextInt();

    System.out.println("\n--- Character Created! ---");
    System.out.println("Name: " + name); 
    System.out.println("Level: " + level);
  }
}
Output (after user enters their details):
--- Create Your Character ---
Enter your character's name: Kael
Enter your character's level: 5

--- Character Created! ---
Name: Kael
Level: 5
