Java's Built-in Calculator: The `Math` Class

Java provides a built-in `Math` class that contains many useful methods for performing mathematical calculations. You don't need to import it; it's always available.

Finding the Highest or Lowest Value

The `Math.max()` and `Math.min()` methods are simple but powerful. They take two numbers and return the one that is larger or smaller, respectively.

int damageFromFire = 45;
int damageFromPoison = 30;
int highestDamage = Math.max(damageFromFire, damageFromPoison);

System.out.println("The highest damage dealt was: " + highestDamage);
Output:
The highest damage dealt was: 45

Square Roots and Absolute Values

The `Math.sqrt()` method calculates the square root of a number, which is useful for tasks like calculating distance. `Math.abs()` returns the absolute (non-negative) value of a number.

// For example, finding the distance between two points (simplified)
double distance = Math.sqrt(144);
int scoreChange = -25;
int absoluteChange = Math.abs(scoreChange);

System.out.println("Distance to target: " + distance);
System.out.println("Score changed by: " + absoluteChange);
Output:
Distance to target: 12.0
Score changed by: 25

Generating Random Numbers

The `Math.random()` method is incredibly useful for games and simulations. It returns a random `double` value between 0.0 and (but not including) 1.0.

To get a random whole number within a specific range, you can use a simple formula.

// Simulate a dice roll (1 to 6)
int diceRoll = (int)(Math.random() * 6) + 1;

// Calculate a random critical hit chance (0 to 100)
int critChance = (int)(Math.random() * 101);

System.out.println("You rolled a: " + diceRoll);
System.out.println("Critical Hit Chance: " + critChance + "%");
Output (will be random):
You rolled a: 4
Critical Hit Chance: 87%
