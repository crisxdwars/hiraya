Making Decisions in Code

Conditional statements allow your program to make decisions. They execute a piece of code only if a certain condition is true.

Java uses standard logical conditions to form these decisions:

  • `if` is used to execute a block of code if a condition is true.
  • `else` is used to execute a different block of code if the condition is false.
  • `else if` is used to check a new condition if the first one was false.

The `if` Statement

The `if` statement is the most basic control flow statement. It runs the code inside its curly braces `{}` only when its condition is `true`.

int score = 85;
if (score >= 75) {
  System.out.println("Congratulations, you passed the exam!");
}
Output:
Congratulations, you passed the exam!

The `else` Statement

The `else` statement provides a secondary path for your code. It runs only when the `if` condition is `false`.

int number = -5;
if (number >= 0) {
  System.out.println("The number is positive.");
} else {
  System.out.println("The number is negative.");
}
Output:
The number is negative.

The `else if` Statement

When you need to check multiple conditions in a sequence, you can use `else if`. Java will test each condition in order until one is true.

int grade = 88;
if (grade >= 90) {
  System.out.println("Excellent! You got an A.");
} else if (grade >= 80) {
  System.out.println("Good job! You got a B.");
} else {
  System.out.println("You can do better.");
}
Output:
Good job! You got a B.
