Performing Actions with Operators

Operators are special symbols that perform operations on variables and values. They are the action words of programming, allowing you to do everything from simple math to complex logical checks.

Arithmetic Operators: The Calculator

These are the basic math operators you're familiar with. They are essential for tasks like calculating scores, damage, or remaining inventory.

int score = 100;
int bonus = 50;
int totalScore = score + bonus; // totalScore is 150

int enemyHealth = 100;
int swordDamage = 35;
int remainingHealth = enemyHealth - swordDamage; // remainingHealth is 65

System.out.println("Final Score: " + totalScore);
System.out.println("Enemy Health: " + remainingHealth);
Output:
Final Score: 150
Enemy Health: 65

Assignment Operators: Storing Values

The most basic assignment operator is `=`, which assigns a value to a variable. There are also shorthand operators like `+=` and `-=` that modify a variable's existing value.

int arrows = 50;
arrows -= 10; // Player used 10 arrows. This is the same as: arrows = arrows - 10;
System.out.println("Arrows remaining: " + arrows);
Output:
Arrows remaining: 40

Comparison Operators: Asking Questions

Comparison operators compare two values and return a `boolean` (`true` or `false`). They are the foundation of decision-making in your code.

int playerLevel = 10;
int requiredLevel = 15;
boolean canUseSword = playerLevel >= requiredLevel;

System.out.println("Can use legendary sword? " + canUseSword);
Output:
Can use legendary sword? false
