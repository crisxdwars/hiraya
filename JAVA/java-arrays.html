Storing Multiple Items: The Inventory

So far, we've used variables that can only hold one value at a time. But what if you need to store a list of items, like a player's inventory or a list of high scores? For this, you use an **array**.

An array is like a backpack with a fixed number of slots. You can store multiple values of the **same type** in a single variable.

Declaring and Initializing an Array

To create an array, you define the data type followed by square brackets `[]`. You can initialize it with a set of starting values inside curly braces `{}`.

// Creating an inventory with three items
String[] inventory = {"Sword", "Shield", "Health Potion"};

// Creating a list of high scores
int[] highScores = {1500, 1250, 1100, 950, 800};

Accessing Items by Index

Each slot in an array has a numbered position called an **index**. The most important rule to remember is that Java starts counting from **zero**. The first item is at index 0, the second is at index 1, and so on.

You access an item by using its index inside square brackets.

String[] inventory = {"Sword", "Shield", "Health Potion"};

// Get the item from the first slot (index 0)
String equippedWeapon = inventory[0];

System.out.println("Player equipped the " + equippedWeapon);
Output:
Player equipped the Sword

Changing an Item and Checking Length

You can change the value of an item by accessing its index. To find out how many slots are in your array, you can use the `length` property.

String[] inventory = {"Sword", "Shield", "Health Potion"};

// The player uses the potion and finds a better one
inventory[2] = "Super Potion";

System.out.println("The third item is now: " + inventory[2]);
System.out.println("Inventory size: " + inventory.length);
Output:
The third item is now: Super Potion
Inventory size: 3

Looping Through an Array

Arrays and loops are a perfect match. You can use a `for` loop to go through every item in an array, which is great for displaying a list or checking each item.

String[] questLog = {"Defeat the Dragon", "Find the Lost Sword", "Deliver the Letter"};

System.out.println("--- Current Quests ---");
for (String quest : questLog) {
  System.out.println("- " + quest);
}
Output:
--- Current Quests ---
- Defeat the Dragon
- Find the Lost Sword
- Deliver the Letter
