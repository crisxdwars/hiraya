Displaying Information

The most common way to see what your program is doing is to print messages to the console. This is essential for debugging and providing feedback to the user.

The `System.out.println()` method is your primary tool for this. It prints the given text or value and then moves the cursor to a new line.

System.out.println("System booting up...");
System.out.println("Player health: 100");
System.out.println("Mission objective: Find the hidden temple.");
Output:
System booting up...
Player health: 100
Mission objective: Find the hidden temple.

`print()` vs. `println()`

If you want to print multiple pieces of information on the same line, you can use the `print()` method. It works just like `println()` but doesn't automatically start a new line.

System.out.print("Loading assets... ");
System.out.print("Done.");
Output:
Loading assets... Done.
