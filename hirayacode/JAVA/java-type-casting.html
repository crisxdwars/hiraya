Changing a Variable's Type

Type casting is the process of converting a variable from one data type to another. Think of it like moving water between different sized cups.

Widening Casting: Small Cup to Big Cup

Widening casting happens when you move a value from a smaller data type to a larger one (e.g., from an `int` to a `double`). This is safe because no information will be lost, so Java does it automatically.

// A player's damage is a whole number
int playerDamage = 50;

// But the final damage calculation needs decimals for modifiers
double finalDamage = playerDamage; // Java automatically converts the int to a double

System.out.println("The final damage is: " + finalDamage);
Output:
The final damage is: 50.0

Narrowing Casting: Big Cup to Small Cup

Narrowing casting is when you move a value from a larger type to a smaller one (e.g., from a `double` to an `int`). This can be risky because you might spill some data (lose the decimal part). You must do this manually by telling Java you accept the risk.

// The precise weight of an item is a double
double itemWeight = 2.85;

// But our inventory can only store whole numbers for weight
int inventoryWeight = (int) itemWeight; // We manually cast it to an int

System.out.println("The item's weight in the inventory is: " + inventoryWeight);
Output:
The item's weight in the inventory is: 2

Notice that the `.85` was lost in the conversion. This is the "spill" you have to be careful about with narrowing casting.
