The Concept of Truth

In programming, we often need to check if something is true or false. Is the user logged in? Is the game over? Does the file exist?

To handle these "yes or no" scenarios, Java provides the `boolean` data type. It can only hold two possible values: `true` or `false`.

Declaring Boolean Variables

You declare a boolean variable using the `boolean` keyword. It's common to name boolean variables with words like "is", "has", or "can" to make the code readable.

boolean isLoggedIn = true;
boolean isGameOver = false;
boolean hasPermission = true;

System.out.println("User is logged in: " + isLoggedIn);
System.out.println("Game over: " + isGameOver);
Output:
User is logged in: true
Game over: false

Boolean Expressions

A boolean expression is any piece of code that results in a `true` or `false` value. The most common way to create them is by using comparison operators like `>` (greater than) or `==` (equal to).

int userLevel = 10;
int requiredLevel = 5;

boolean canEnterDungeon = userLevel >= requiredLevel;

System.out.println("Can the user enter the dungeon? " + canEnterDungeon);
Output:
Can the user enter the dungeon? true

These expressions are the foundation of decision-making in Java and are used heavily in `if` statements, which you'll learn about next.
