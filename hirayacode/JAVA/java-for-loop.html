The Assembly Line of Loops

When you know exactly how many times you need to repeat a task, the `for` loop is the perfect tool. Think of it as setting up an assembly line: you know it needs to process exactly 10 items before stopping.

The `for` loop combines the three key parts of a loop into one clean line: initialization, condition, and increment.

The Classic `for` Loop

The classic `for` loop has three statements inside its parentheses:

  1. Initialization: A counter variable is created and set to a starting value (e.g., `int i = 0`). This runs only once.
  2. Condition: The loop continues as long as this condition is true (e.g., `i < 5`).
  3. Increment: A statement that runs after each loop iteration, usually to increase the counter (e.g., `i++`).

// Crafting 5 health potions
for (int i = 1; i <= 5; i++) {
  System.out.println("Crafting potion #" + i);
}
System.out.println("All potions have been crafted!");
Output:
Crafting potion #1
Crafting potion #2
Crafting potion #3
Crafting potion #4
Crafting potion #5
All potions have been crafted!

The Enhanced `for` Loop (For-Each)

When you need to loop through all the elements in a collection, like an array, there's an even simpler syntax called the "for-each" loop. It handles the indexing and condition for you automatically.

Its syntax is: `for (type variableName : arrayName)`.

// Checking quests in a quest log (array)
String[] questLog = { "Defeat the Dragon", "Find the Lost Sword", "Deliver the Letter" };

System.out.println("--- Current Quests ---");
for (String quest : questLog) {
  System.out.println("- " + quest);
}
Output:
--- Current Quests ---
- Defeat the Dragon
- Find the Lost Sword
- Deliver the Letter
